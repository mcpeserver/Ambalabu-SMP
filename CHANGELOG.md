# Catatan Perubahan (Changelog)

Semua perubahan penting pada proyek **AMBALABU SMP Landing Page** akan didokumentasikan dalam file ini.

## [1.0.0] - 2026-07-01
### Ditambahkan (Added)
- Desain Landing Page modern dengan visual Minecraft ramah dan menyenangkan.
- Skema warna hangat (cokelat, kuning emas, hijau rumput, putih, dan oranye).
- Informasi Server IP Java & Bedrock Edition dengan fitur "Satu Klik untuk Menyalin" (Click to Copy).
- Daftar Aturan Resmi Server AMBALABU SMP.
- Daftar Rank Server resmi beserta rincian harganya.
- Widget Detektor Koneksi (Simulated Latency Meter) interaktif.
- Struktur SEO lengkap: meta description, Open Graph tags, Schema.org JSON-LD microdata, dan manifest.webmanifest.
- Dokumen pendukung repository lengkap (`LICENSE`, `README.md`, `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`, `CHANGELOG.md`).
- Halaman penanganan error `404.html` berdesain konsisten.
- Penanda hak cipta pengembang (Developed by RAN DEV) di header dan footer.
